# 🎓 WeibanPro 微伴助手

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/status-active-success)

**WeibanPro** 是一款专为微伴网课平台打造的现代化智能辅助工具。本项目基于 Python 和 PyQt5 开发，拥有全新的现代化 UI 界面，集成了**全自动刷课**、**智能题库答题**、**验证码自动识别**等核心功能，旨在帮助用户高效完成繁琐的学习任务。

---

## ✨ 核心亮点

### 🎨 现代化交互体验
- **卡片式布局**：清晰的信息层级，左侧配置，右侧监控。
- **动态交互**：精心设计的果冻按钮动画与无边框弹窗体验。
- **实时监控**：直观的进度条展示与详细的运行日志反馈。

### 🤖 智能化学习流程
- **全自动接管**：一键启动后，自动处理登录、选课、看视频、答题全流程。
- **智能验证码**：内置本地 AI 模型 (`ddddocr`)，毫秒级识别登录验证码。
- **精准答题**：
    - 内置海量题库 (`result.json`)。
    - 自动匹配最优答案。
    - 支持**错题阈值控制**，智能模拟真实答题准确率，规避风控。

### 🛡️ 安全与隐私
- **本地运行**：所有逻辑均在本地执行，不上传用户账号密码。
- **行为模拟**：智能随机延迟与视频观看倒计时，模拟真人操作轨迹。

## 📖 使用详细步骤

1.  **登录设置**
    *   在左侧面板输入您的**微伴账号**和**密码**。
    *   **关键**：输入完整的**学校中文名称**（如“凯里学院”），否则无法匹配学校接口。
    *   点击 **“登录获取课程”** 按钮，系统将自动登录并加载您的课程列表。

2.  **任务配置**
    *   **选择课程**：从下拉菜单中选择您需要学习的课程。
    *   **考试时长**：设置自动答题的耗时（建议 `240` - `300` 秒），避免交卷过快。
    *   **允许错题**：设置允许答错的题目数量（建议 `3` - `5` 题），模拟真实水平。

3.  **开始运行**
    *   点击 **“开始任务”** 按钮。
    *   程序将自动进入工作状态，您可以在右侧的 **“学习监控中心”** 看到：
        *   当前正在学习的章节
        *   视频观看倒计时
        *   答题结果统计

## ❓ 常见问题 (FAQ)

**Q: 为什么登录失败？**
A: 请检查账号密码是否正确。最常见的原因是**学校名称输入不完整**，请务必输入学校的全称（不要用简称）。

**Q: 题库里没有我的题怎么办？**
A: 程序内置了在线搜索逻辑，如果本地题库 (`result.json`) 未找到，会自动尝试使用 AI 或随机策略。同时，程序会自动将新遇到的题目收集并更新到本地题库中。

**Q: 运行过程中可以关闭窗口吗？**
A: 不可以。关闭窗口会终止当前的刷课任务。您可以将其最小化到后台运行。
